# Project: PAC Crowdwatch

MVP Version 1

Crowd Management System for Pat-A-Cat with the following features:

1. Visitor (Individual / Group) Check-In
2. Cash / PayNow collection
3. Dynamic QR generation (with embedded UEN + amount for convenience)
4. Drinks allocation tracking
5. End time tracking
6. Extension of hours (including individuals of a group)
7. Admin management page with following key functionalities:
   a. Add / delete products
   b. Toggle products activity / inactivity
   c. Edit prices
   d. Change admin password
8. Ability to view Daily Total Income by breakdown:
   a. Cash / PayNow
   b. Groups / Individuals
9. New Sale
   a. Ability to sell treats
   b. Ability to take in donations
